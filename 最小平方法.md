## 最小平方法 - 實作

### *生成一個Data*

取樣本數總共有100個
x1的函數為 50 + 30*隨機數值            
y的函數為 135 + 0.5 * x1 + 3 * 隨機數值          
x可以當作體重                
y可以當作身高

```Python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(1) # 保證跑出來數字跟下面一樣

m = 100 # 樣本數
x1 = 50 + 30 * np.random.rand(m, 1) # m*1的array
y = 135 + 0.5 * x1 + 3 * np.random.randn(m, 1) # 後面為隨機誤差

x1[:5] # 當作體重
y[:5] # 當作身高
```
